Intervention Point 1
Elevated IRT result
Communication when all testing complete — ideally first week of life. Primary communicator: pediatrician/PCP. EHR may show partial results before family is informed.
Intervention Point 2
DNA panel result
PCP informs family of variant result and next steps. PCP provides CF center contact and orders sweat test. Some states also notify CF center directly.
Intervention Point 3
Sweat test scheduled
PCP refers to CF center for sweat test. Standard: ≥1 variant. Some states (CA, NY, FL, others) only refer with ≥2 variants.

DNA panel categories — how the four differ source: 2025 CF NBS State Algorithm Data (Marci/Yvonne → K. Eilbeck) · "CFTR panel category" column + "Panel Map" sheet
D · CFTR Panel, 39–75 variants
Fixed panel on standard (non-sequencing) chemistry. Reports only the panel's variants. Largest group.
C · NGS Panel
A CFTR panel run by next-generation sequencing (NGS), but only a predefined list is ever reported — no novel or VUS/VVCC variants. Tiered programs live here too: DE & PA run a 43-variant Revvity screen and report from a 338-variant Revvity list — larger, but still a fixed list.
B · Panel → Sequencing
First-tier panel reflexes to true CFTR sequencing that can report novel + interpreted variants (CFTR2, VUS, VVCC). CA, CT, FL, MD, TN.
A · CFTR2 (+)
Full CFTR sequencing interpreted against the CFTR2 database (≈978–1100 variants). KS, MT, NY, OH, VA, WI.
PA vs FL — both are tiered (a smaller panel runs first). The source separates them by what the second tier can report: FL reflexes to open sequencing that reports novel / VUS / VVCC variants (group B), while PA's second tier is a larger but still fixed 338-variant list (group C). The source "Panel Map" keeps PA in C. To reclassify PA, edit its pcat_short / cluster / panel_group in the STATES data — see the how-to-edit notes.
