# CF Newborn Screening — State Workflow Tree

**[View the tree →](https://klanaghen.github.io/REPO-NAME/)**

An interactive map of how all 51 U.S. cystic fibrosis newborn screening programs move an infant from the first IRT blood spot through to a sweat-test referral.

## Why it exists

This is the reference document behind a chatbot being developed to guide families through the CF newborn screening process — the stretch between "your baby's screen was abnormal" and a definitive answer. The bot has to branch by state, because states genuinely differ in what happens next. Every branch in this tree becomes a sentence said to a parent, so it needs to be right before anything gets scripted.

## What you're looking at

- **51 state programs**, grouped into 11 distinct workflows
- **Three intervention points** — elevated IRT → DNA result → sweat-test referral
- **Hover any state** for its IRT cutoffs, panel technology, sweat-test trigger, notification practices, and the source behind each claim
- **Filters** across the top isolate two-sample states, panel categories, and sweat-test triggers

Each state pill carries a small colored square marking how well-sourced it is — spreadsheet, expert communication, web-verified, or unverified.

## Feedback wanted

This is a working draft and parts of it likely incorrect. If your state is misrepresented, or a category looks off, please say so — corrections are the point of sharing it.

## Keeping Up-to-Date
The largest challenge moving forward will be reporting testing an workflow changes. Encouraging states' individual resposibility to report said changes will make the bot development process and maintenance much easier.
